# 1C
Пересчет цены общего модуля
В модуле формы документа "Реализация товаров и услуг" добавлены клиентские процедуры для перерасчёта цены строки.
•	Создана процедура ПересчитатьСуммуСтроки(), вызываемая при изменении номенклатуры. Если номенклатура заполнена, выполняется вызов ЦеныВызовСервера.ЦенаНаДату для получения актуальной цены.
•	В обработчике события ПриИзменении для поля номенклатуры добавлен вызов процедуры ПриИзмененииНоменклатуры, которой передаются текущие данные таблицы цен.

![Пересчет цены в РТУ](https://github.com/USER-404101/1C/blob/main/Пересчет%20цены%20в%20РТУ.png)

•	Добавлен общий модуль ЦеныСервер (флажки: Сервер). В него помещена экспортная функция ЦенаНаДату(Номенклатура, Дата), выполняющая запрос для получения цены срезом последних по указанной дате и возвращающая результат.
•	Добавлен общий модуль ЦеныВызовСервера (флажки: Сервер, Внешнее соединение, Вызов сервера), содержащий функцию ЦенаНаДату для вызова серверной функции из клиентского кода форм.
